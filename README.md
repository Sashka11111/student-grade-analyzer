# Student Grade Analyzer

Професійний інструмент для автоматизованого аналізу академічної успішності студентів на основі Excel-відомостей.

## 🌟 Основні можливості

- **Точні розрахунки**: Обчислення розподілу оцінок за шкалою ECTS (A, B, C, D, E, FX, F) з точністю до 1 знака після коми.
- **Корекція округлення**: Використання алгоритму *Largest Remainder Method* для гарантування того, що сума відсотків завжди дорівнює рівно 100.0%.
- **Шаблони Excel**: Генерація готового до заповнення порожнього шаблону відомості.
- **Професійний друк**: Спеціально налаштований режим друку, який виводить на аркуш лише чисту таблицю з результатами без зайвих елементів інтерфейсу.
- **Експорт даних**: Можливість завантаження результатів аналізу у форматі CSV.
- **Сучасний дизайн**: Преміальний UI з використанням Glassmorphism, іконок Lucide та шрифту Inter.

## 🛠 Технологічний стек

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Обробка Excel**: [SheetJS (xlsx.full.min.js)](https://github.com/SheetJS/sheetjs).
- **Іконки**: [Lucide Icons](https://lucide.dev/).
- **Шрифти**: Google Fonts (Inter).

## 🚀 Як користуватися

1. **Завантаження шаблону**: Натисніть кнопку "Завантажити шаблон", щоб отримати Excel-файл з правильною структурою.
2. **Підготовка даних**: Заповніть шаблон ПІБ студентів та їхніми оцінками.
3. **Аналіз**: Перетягніть ваш файл у зону завантаження або оберіть його через провідник.
4. **Налаштування**: Перевірте діапазон рядків та кількість предметів (за замовчуванням встановлено 48). Натисніть "Виконати аналіз".
5. **Результат**: Отримайте детальну таблицю з розподілом ECTS.
6. **Друк/Збереження**: Натисніть "Друкувати таблицю" для паперового звіту або "Зберегти CSV" для подальшої обробки даних.

## 📂 Структура проекту

- `index.html` — основна розмітка та структура сторінки.
- `style.css` — стилі оформлення, анімації та налаштування друку.
- `script.js` — логіка обробки файлів, математичні розрахунки та UI-взаємодія.
